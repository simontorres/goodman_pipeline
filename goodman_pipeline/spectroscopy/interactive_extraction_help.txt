This is the interactive target extraction tool.

The goal is to allow the use of the regular functions but with more
control on how they work.

The application react to certain events such as clicks and keystrokes:

Trace Manipulation:

  On the 2D Image
  Middle button Click:
    Selects the current pixel row as the center of a trace

General Actions:

  h:
    Displays the help windows.
  ctrl+p:
    Enters preview mode.
  ctrl+q:
    Closes interactive window.


Preview Mode:
  In preview mode a sample of the width of "extraction width" is extracted.
  But it does not track the trace.
